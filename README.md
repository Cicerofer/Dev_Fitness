# Dev_Fitness
DevFitness - Jornada .NET Direto ao Ponto

 #### Foi desenvolvida uma aplicação Console completa, utilizando os conceitos de fundamentos de C#, Listas e Programação Orientada a Objetos e uma API REST completa de gerenciamento de refeições e informações calóricas delas, junto com os dados do usuário
  #### Tecnologia utilidas nesse projéto:
  
 
- ASP.NET Core com .NET 5
- Entity Framework Core
- Swagger
- AutoMapper
- Injeção de Dependência
- Programação Orientada a Objetos
